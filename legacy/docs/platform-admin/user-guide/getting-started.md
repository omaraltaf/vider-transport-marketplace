# Getting Started with Platform Admin Dashboard

## Initial Setup

### First Time Login

1. **Access the Dashboard**
   - Navigate to: `https://admin.vider.no`
   - Use your provided admin credentials
   - Click "Sign In"

2. **Multi-Factor Authentication Setup**
   - Download an authenticator app (Google Authenticator, Authy)
   - Scan the QR code displayed
   - Enter the 6-digit verification code
   - Save your backup codes securely

3. **Profile Setup**
   - Complete your profile information
   - Set notification preferences
   - Configure dashboard layout

### Dashboard Overview

The main dashboard provides:
- **Quick Stats**: Key platform metrics at a glance
- **Recent Activity**: Latest admin actions and system events
- **Alerts Panel**: Security alerts and system notifications
- **Quick Actions**: Shortcuts to common tasks

### Essential First Steps

1. **Review Pending Items**
   - Check company verification queue
   - Review security alerts
   - Monitor content moderation queue

2. **Familiarize with Key Metrics**
   - Total users and companies
   - Revenue and transaction volumes
   - Security event counts

3. **Set Up Notifications**
   - Configure email alerts
   - Set up mobile notifications
   - Customize alert thresholds

### Keyboard Shortcuts

- `Ctrl + /`: Open command palette
- `Ctrl + D`: Go to dashboard
- `Ctrl + U`: Go to users
- `Ctrl + C`: Go to companies
- `Ctrl + A`: Go to analytics
- `Esc`: Close modals/panels

### Getting Help

- **In-App Help**: Click the "?" icon in the top right
- **Documentation**: Access from the help menu
- **Support**: Contact support@vider.no
- **Training**: Schedule training sessions with the team
